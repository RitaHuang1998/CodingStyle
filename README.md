# CodingStyle

## Python Coding Style
### Semicolon
```python
# No
a = 1;
b = 1; c = a + b

# Yes
a = 1
b = 1
c = a + b
```
### line length
* No more than 80 characters per line

### Brackets
```python
# No
return (x, y)
if (x and y): ...


# Yes
return x, y
if x and y
```
